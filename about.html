Acerca de Nosotros

Más acerca de la página

Descubre nuestra plataforma web diseñada para ayudarte a encontrar el lugar perfecto cerca de la universidad. Somos una guía completa de restaurantes donde puedes explorar opciones, leer reseñas de otros usuarios y realizar reservaciones fácilmente.

Nuestra página está integrada con Google Maps para que puedas ubicar con precisión cada establecimiento. Este proyecto fue desarrollado utilizando un flujo de trabajo colaborativo con Git y GitHub, gestionando las tareas mediante issues y organizando el desarrollo en ramas específicas para asegurar la calidad y facilitar las actualizaciones.
